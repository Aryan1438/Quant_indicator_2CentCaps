Aryan Price Action Framework

This report is generated by the Python backtesting engine and is designed to be paired with TradingView exports, institutional order-block logic, liquidity sweeps, and multi-timeframe trend analysis.

What this report contains

Equity curve, performance metrics, trade log, and execution markers are exported into the reports directory.

Run command

python python-backtester/main.py

Research context

The framework evaluates BOS, CHoCH, CHoCH+, order-block confirmation, liquidity sweeps, FVG confirmation, and higher-timeframe alignment. It is intended for internship-grade research presentation rather than retail-style indicator use.
