# Internship Report

## Project Title

Aryan Price Action Framework

## Objective

Build a professional quant research system that studies Smart Money Concepts, institutional order blocks, liquidity sweeps, and multi-timeframe market structure in a way that is suitable for a quant research internship portfolio.

## Deliverables

- Pine Script v5 strategy for chart research and alerts
- Python backtesting engine for performance analysis
- Streamlit dashboard for analytics and trade review
- Research notes and strategy documentation
- Reproducible project structure for GitHub presentation

## Research Summary

The framework is designed around a confluence hypothesis: structure breaks are more reliable when they occur in the direction of the higher-timeframe trend and are supported by liquidity events and order-block context.

## Performance Metrics

The backtesting pipeline measures:

- Win rate
- Profit factor
- Sharpe ratio
- Sortino ratio
- Max drawdown
- CAGR
- Expectancy
- Average reward-to-risk
- Monthly returns
